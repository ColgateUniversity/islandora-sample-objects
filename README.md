# Islandora Sample Objects

Sample set of objects and corresponding metadata. Examples are provided in raw and zipped formats, allowing for quicker bulk ingestion into Islandora.

## Content Models Provided

* AUDIO
* BASIC_IMAGE
* BOOK
* LARGE_IMAGE
* PDF
* VIDEO
